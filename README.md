## Created by:

## Technologies
* React.js
* Netlify

## Prerequisites
* Installed Node.js https://nodejs.org/en/download/

## Running the App
* Install all the packages in the root directory
  ```
    npm install
  ```
* Running the website in http://localhost:3000/
  ```
    npm start
  ```
